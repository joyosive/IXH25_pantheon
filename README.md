## Quick Start

### Prerequisites

- Node.js 16+
- Foundry
- Circom 2.0+

### Installation

```bash
# Install dependencies
npm install

# Install Foundry dependencies
forge install
```

### Build & Test

```bash
# Compile Solidity contracts
forge build

# Run tests
forge test

# Compile ZK circuits (when circom is available)



```
